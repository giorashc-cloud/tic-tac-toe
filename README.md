## About

This is a straightforward implementation of the classic tic-tac-toe game. It features:

- Interactive 3x3 game board
- Two-player gameplay (X and O)
- Win detection for rows, columns, and diagonals
- Draw detection when the board is full
- Game reset functionality
- Built with React, TypeScript, and Vite
- Includes unit tests with Vitest

## How to Run

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

Then open your browser to the URL shown in the terminal (typically `http://localhost:5173`).

### Build

Create a production build:

```bash
npm run build
```

### Testing

Run the tests:

```bash
npm test
```

Run tests with UI:

```bash
npm run test:ui
```
